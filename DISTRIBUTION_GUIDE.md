## 🚀 用户安装步骤

### 1. 下载并打开DMG
- 双击 `CTCClick-20251029-020541.dmg` 文件
- DMG将自动挂载并显示安装界面

### 2. 安装应用
- 将 `CTCClick.app` 拖拽到 `Applications` 文件夹
- 等待复制完成

### 3. 首次启动
- 在Applications文件夹中找到CTCClick
- **重要**: 右键点击应用，选择"打开"（不要直接双击）
- 在弹出的安全提示中点击"打开"

### 4. 启用Finder扩展
由于这是未签名的应用，Finder扩展需要手动启用：

#### 方法一：使用脚本（推荐）
```bash
# 下载并运行启用脚本
./enable_finder_extension.sh
```

#### 方法二：手动启用
1. 打开"系统偏好设置"
2. 选择"扩展"
3. 在左侧选择"Finder扩展"
4. 勾选"CTCClick"

#### 方法三：如果扩展不显示
某些情况下扩展可能不会立即显示，请：
1. 重启电脑
2. 或运行修复脚本：`./fix_finder_extension.sh`

## 🔧 故障排除

### Finder扩展无法启用
如果按照上述步骤仍无法启用扩展，请运行：
```bash
./fix_finder_extension.sh
```

该脚本会：
- 重新签名扩展
- 清理并重新注册扩展
- 重启相关服务

### 应用无法打开
- 确保已通过右键菜单选择"打开"
- 检查"系统偏好设置 > 安全性与隐私 > 通用"中是否有相关提示

### 扩展权限问题
- 在"系统偏好设置 > 安全性与隐私 > 隐私"中检查相关权限
- 确保CTCClick有必要的文件访问权限
